# OVERSIZE shop

MERN stack (MongoDB, ExpressJS, React and Node.JS)

## installation

Before you start you need to clone the itn repository and run itn core services on your local machine.
Run in the root directory of the cloned repository:

## Run Backend

```shell
cd server
```

```shell
npm install
```

```shell
npm start
```

## Run Frontend

```shell
cd web
```

```shell
npm install
```

```shell
npm run dev
```
